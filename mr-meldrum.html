Field: Software Engineering

How computer science is used in their field:

Computer science is the cornerstone of software engineering. It is a tool that we use when building software solutions to help guide our decision process.

How computer science could be used in their field:

Many of the difficult computer science problems have been solved. With shareable libraries, everyone is able to benefit from the hard work of a few dedicated computer scientists. This is a great opportunity for the use of machine learning in the software development process. At the moment, IDE’s are sophisticated and are able to make informed suggestions based on best practices.

With the aid of machine learning, IDE’s could become sophisticated enough to act as a peer programmer, making suggestions on algorithms and general solution architecture.

How computer science has been used in their field:

For many years, software engineers were forced to reinvent the wheel because there were no suitable shared libraries and there was no easy mechanism for sharing code. As networking technologies evolved, people were able to more easily share code libraries. Also, with the evolution of higher level programming languages, it has become easier to represent data structures and algorithms in a way that is easier for humans to understand and interact with.

There are many more tools available for the average software engineer, so they are able to achieve more with less effort.
